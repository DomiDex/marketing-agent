## Free Tool Strategy Command

### Instructions

1. **Invoke the free-tool-strategy skill** to guide this interaction
2. **Gather context** about the tool idea and marketing goals
3. **Follow the skill's methodology** to plan the tool
4. **Save all outputs** to: `output/free-tool-strategy/`

### Output Requirements

**Directory**: `output/free-tool-strategy/`
**Filename**: `tool_{descriptor}_{YYYY-MM-DD}.md`

Example: `tool_roi-calculator_2026-01-22.md`
